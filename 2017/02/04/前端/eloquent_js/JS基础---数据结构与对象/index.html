JS基础 - 数据结构与对象

发表于 2017-02-04 |

数组

  • 数组中的元素可以是不同类型的
  • 数组下标以0开始

属性

  • 获取属性的方法有两种：
    • obj.pro: 点号后面的属性名必须是有效的属性名
    • obj[pro]: 括号中可以是任何形式的表达式，表达式先经过计算，再被当作对象的属性获取相应的属性信息
  • 数组可以认为是属性全部为数字的对象
  • 值为函数的属性被称为是方法
  • 对象的属性通过key: value的形式指定，尝试访问不存在的属性会返回undefined, 可以通过=进行新增或更新属性操作，delete操作符可以将对象的属性删除
